# Infobip [![npm version](https://badge.fury.io/js/infobip.svg)](https://badge.fury.io/js/infobip)

Infobip API Client for Node.js using [Infobip API](https://dev.infobip.com)


## Installation

```bash
$ npm install --save infobip
```
